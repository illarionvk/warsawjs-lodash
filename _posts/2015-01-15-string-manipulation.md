## String Manipulation

{% highlight javascript linenos %}
_.startCase('+Hello++world+');
// → Hello World
{% endhighlight %}

{% highlight javascript linenos %}
_.camelCase('+Hello++world+');
// → helloWorld
{% endhighlight %}

{% highlight javascript linenos %}
_.snakeCase('+Hello++world+');
// → hello_world
{% endhighlight %}
